### What I learned

During this project, i improved my ability to structure a web page using sematic HTML elements like '<main>' and '<section>', which helps organize content and improves accessibility.
Iused Flexbox to align and structure different parts of the layout, making the design clean and responsive.
I also learned how to use HTML tables to display structured data clearly, such as nutritional information

Example:

